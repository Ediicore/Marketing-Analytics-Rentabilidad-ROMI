# 📈 Optimización de Marketing Analytics - Showz
### Análisis de Rentabilidad (ROMI) y LTV para Optimización de Presupuesto SaaS

---

## 📝 Contexto del Proyecto
En este proyecto, realicé un análisis profundo de las métricas de marketing para una plataforma de espectáculos (Showz). El desafío era identificar qué canales de adquisición de clientes estaban generando pérdidas y cuáles tenían el mayor potencial de crecimiento a largo plazo.

## 🎯 Objetivos Clave
* **Análisis de Cohortes:** Evaluar el comportamiento de los usuarios según su mes de registro.
* **Cálculo de Rentabilidad:** Determinar el ROMI (Retorno de Inversión en Marketing) por canal.
* **Predicción de Valor:** Calcular el LTV (Lifetime Value) para priorizar inversiones.

## 🛠️ Stack Tecnológico
| Herramienta | Uso Principal |
| :--- | :--- |
| **Python (Pandas)** | Procesamiento de datos de ventas y registros. |
| **Cohort Analysis** | Segmentación de usuarios por tiempo y comportamiento. |
| **Business Metrics** | Cálculo de CAC, LTV y ROMI. |

## 📊 Resultados e Impacto de Negocio
El análisis permitió tomar decisiones estratégicas basadas en evidencia financiera:
* 💰 **Eficiencia Presupuestaria:** Se identificaron canales con ROMI negativo, permitiendo la redistribución del presupuesto hacia fuentes más rentables.
* 🚀 **Crecimiento de LTV:** Identificación de los segmentos de clientes con mayor ciclo de vida, optimizando la retención.
* 📊 **Dashboards de Control:** Implementación de visualizaciones para monitorear el costo de adquisición (CAC) en tiempo real.

## 📂 Contenido del Repositorio
* `Marketing-Analítica-Rentabilidad-ROMI.ipynb`: Notebook detallado con el proceso de análisis y gráficas.
* `README.md`: Resumen ejecutivo del proyecto.

# 📈 Optimización de Análisis de Marketing - Showz
### Análisis de Rentabilidad (ROMI) y LTV para Optimización de Presupuesto SaaS

---

## 📝 Contexto del Proyecto
En este proyecto, realicé un análisis profundo de las métricas de marketing para la plataforma de espectáculos **Showz**. El desafío principal fue identificar qué canales de adquisición de clientes estaban generando pérdidas y cuáles presentaban el mayor potencial de rentabilidad a largo plazo, permitiendo una toma de decisiones basada en datos financieros.

## 🎯 Objetivos Clave
* **Análisis de Cohortes:** Evaluar el comportamiento y retención de los usuarios según su mes de registro.
* **Cálculo de Rentabilidad:** Determinar el **ROMI** (Retorno de Inversión en Marketing) desglosado por canal de adquisición.
* **Predicción de Valor:** Calcular el **LTV** (Lifetime Value) y el **CAC** (Costo de Adquisición de Clientes) para optimizar el presupuesto.

## 🛠️ Stack Tecnológico
| Herramienta | Uso Principal |
| :--- | :--- |
| **Python (Pandas)** | Procesamiento de grandes volúmenes de datos de ventas y registros (ETL). |
| **Análisis de Cohortes** | Segmentación avanzada de usuarios por tiempo y comportamiento de compra. |
| **Métricas Financieras** | Modelado de fórmulas para CAC, LTV y Retorno de Inversión (ROMI). |

## 📊 Resultados e Impacto de Negocio
El análisis permitió definir estrategias financieras críticas:
* 💰 **Eficiencia Presupuestaria:** Se identificaron canales con ROMI negativo, recomendando la redistribución del presupuesto hacia fuentes un **20% más rentables**.
* 🚀 **Crecimiento de LTV:** Identificación de los segmentos de clientes con mayor ciclo de vida, permitiendo enfocar los esfuerzos de retención en clientes de alto valor.
* 📊 **Control de Costos:** Visualización clara de la relación LTV/CAC para asegurar que la adquisición de clientes sea sostenible financieramente.
