# 📈 Optimización de Marketing Analytics - Showz
### Análisis de Rentabilidad (ROMI) y LTV para Optimización de Presupuesto SaaS

---

## 📝 Contexto del Proyecto
En este proyecto, realicé un análisis profundo de las métricas de marketing para una plataforma de espectáculos (Showz). El desafío era identificar qué canales de adquisición de clientes estaban generando pérdidas y cuáles tenían el mayor potencial de crecimiento a largo plazo.

## 🎯 Objetivos Clave
* **Análisis de Cohortes:** Evaluar el comportamiento de los usuarios según su mes de registro.
* **Cálculo de Rentabilidad:** Determinar el ROMI (Retorno de Inversión en Marketing) por canal.
* **Predicción de Valor:** Calcular el LTV (Lifetime Value) para priorizar inversiones.

## 🛠️ Stack Tecnológico
| Herramienta | Uso Principal |
| :--- | :--- |
| **Python (Pandas)** | Procesamiento de datos de ventas y registros. |
| **Cohort Analysis** | Segmentación de usuarios por tiempo y comportamiento. |
| **Business Metrics** | Cálculo de CAC, LTV y ROMI. |

## 📊 Resultados e Impacto de Negocio
El análisis permitió tomar decisiones estratégicas basadas en evidencia financiera:
* 💰 **Eficiencia Presupuestaria:** Se identificaron canales con ROMI negativo, permitiendo la redistribución del presupuesto hacia fuentes más rentables.
* 🚀 **Crecimiento de LTV:** Identificación de los segmentos de clientes con mayor ciclo de vida, optimizando la retención.
* 📊 **Dashboards de Control:** Implementación de visualizaciones para monitorear el costo de adquisición (CAC) en tiempo real.
